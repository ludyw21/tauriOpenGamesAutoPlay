# 开放世界自动演奏工具

这是一个用于自动演奏MIDI文件的工具，专为开放世界游戏设计，支持多种游戏环境。支持鼠标模拟或者键盘模拟。

## ✨ 功能特点

### 核心功能
- **智能MIDI播放** - 支持任意键模式（最大88键）
- **多音轨支持** - 可选择播放单个或多个音轨
- **效果试听** - 支持配置效果试听，提前确认游戏内播放效果
- **智能移调优化** - 自动计算最优移调方案，减少用户操作
- **自定义按键映射** - 灵活配置游戏按键
- **自定义鼠标映射** - 灵活配置游戏鼠标

### 播放控制
- **播放/暂停/停止** - 完整的播放控制功能
- **进度显示** - 实时显示剩余时间
- **快捷键支持** - 全局快捷键控制播放
- **窗口置顶** - 保持窗口始终可见

### 智能分析
- **音高分析** - 自动检测最高音和最低音
- **超限提示** - 智能提示音符超出范围
- **优化建议** - 基于移调+转位绝对值最小原则提供建议

## 🚀 系统要求

- **操作系统**: Windows 10、macOS 或 Linux
- **权限**: 管理员权限（用于键盘控制）

## 🎮 使用指南

### 基本操作流程
0. **默认设置**
   - 燕云十六声 36键 键盘模拟
   - 可以在设置面板进行灵活设置
   - 支持钢琴88键全键位设置
   - 支持键盘模拟 或 鼠标模拟

1. **选择MIDI文件夹**
   - 点击"选择MIDI文件夹"按钮
   - 选择包含MIDI文件的目录

2. **选择音轨**
   - 在右侧列表中选择要播放的音轨
   - 支持多选音轨同时播放

3. **调整设置**
   - 设置移调（半音）和转位（八度）
   - 查看智能优化建议
   - 点击"效果试听"按钮试听效果

4. **开始播放**
   - 点击"开始模拟"按钮或使用快捷键
   - 实时查看播放进度

### 智能移调功能

程序会自动分析MIDI文件的音高范围，并提供最优移调建议：
- **优化标准**: 移调+转位的绝对值最小
- **优先选择**: 5、6、7三个居中移调值
- **自动计算**: 基于-2到+2八度范围生成组合

**示例**:  
当最高音a³(93)超限50，最低音c(48)未超限时：
- 优化前建议：移调-10，转位0（总和10）
- 优化后建议：移调2，转位-1（总和3）

### 鼠标模拟

鼠标模拟支持鼠标移动到指定位置，然后点击鼠标左键，实现自动演奏。
- 自然的鼠标移动：使用二次贝塞尔曲线生成平滑的鼠标移动路径
- 坐标随机化：目标坐标添加 ±5 像素的随机偏移
- 鼠标模拟不支持延音，自动生成释放事件，不按照mini文件的note_off事件进行释放鼠标点击

### 快捷键

在弹奏界面可使用以下快捷键（默认 可自定义）：

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Alt + -` | 播放/暂停 | 切换播放状态 |
| `Alt + =` | 停止播放 | 停止当前播放 |
| `Alt + ↑` | 上一首 | 切换到上一首MIDI文件 |
| `Alt + ↓` | 下一首 | 切换到下一首MIDI文件 |

## ⚠️ 注意事项

### 管理员权限
- Windows 下需要管理员权限才能正常触发键盘事件
- macOS 下需要授予“辅助功能”权限才能正常触发键盘事件
- Linux 理论上可以执行，未测试

### 游戏设置
- 确保游戏窗口标题正确设置
- 建议在游戏中设置合适的按键映射
- 游戏窗口需要保持在前台

### 配置文件
- 程序会自动创建 `config.json` 保存配置
- 配置文件包含最后访问的目录、窗口置顶状态和主题设置
